## docs/automation/netview — IBM Z NetView for z/OS 6.4

NetView is IBM's network and systems management platform for z/OS. It provides:
- Automated operations via command lists (CLISTs) and NetView REXX
- SNA/VTAM network management (AON — Automated Operations Network)
- IP network management
- Event-driven automation through the NetView automation table
- Cross-domain and cross-system automation via sysplex support

## docs/automation/sa-zos — IBM Z System Automation 4.4

System Automation for z/OS (SA z/OS) is IBM's policy-based automated operations framework. It provides:
- Resource/subsystem lifecycle management (start, stop, recycle, restart)
- Automation policy definition via the Customization Dialog (ISPF)
- Sysplex-wide automation and cross-system dependency management
- Health-based automation and goal-driven policy
- Integration with NetView as its event and command engine
